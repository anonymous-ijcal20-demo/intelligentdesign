# IntelligentDesign

## 安装环境
```bash
conda create -n intelligent python=3.7
conda activate intelligent
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 起服务
```bash
python -m server.entrypoint
```
